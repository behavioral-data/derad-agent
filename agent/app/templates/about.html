Eddie Bot is an AI chatbot created by researchers at the University of Washington studying online social dynamics.

{# ── How it works ── #}
How it works

When Eddie Bot is mentioned in reply to a post, it searches the internet for sources before generating a fact-check of the claim in that post. Each reply links to an evidence dossier showing the sources reviewed and the reasoning behind the assessment.

{# ── Limitations ── #}
Limitations
Because this bot uses AI, responses may contain inaccurate information. All claims should be verified using the source links provided in each response.

The study

Study ID: STUDY00025610

This bot generates fact-checks by retrieving live web evidence (independent fact-checkers, reputable news outlets, and primary sources) and reconciling that evidence against the post. Source reliability is classified using the IFCN signatory list, Wikipedia perennial-sources ratings, and a model-based fallback for uncovered domains. Responses may contain inaccuracies — verify claims using the cited sources.
